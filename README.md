# 🌿 Sreeshanth Reddy Namireddy — The Living Digital Ecosystem Portfolio

A living, breathing digital ecosystem designed to tell your story as an AI Systems & Infrastructure Builder. The portfolio is built using a hybrid HTML + 2D Canvas approach, ensuring 60fps performance, complete mobile responsiveness, and SEO indexing.

---

## 🚀 Key Features

* **The Living Canvas Background (`engine.js`)**: 12+ layers of depth including sky gradients, far mountain ridges, mid hills, tree lines, and water basins that pan vertically and horizontally as you scroll.
* **Procedural Sky Color Dial**: A day/night range slider dynamically transitions the sky through Morning, Noon, Sunset, and Night colors.
* **Boids Flocking (Birds) & Swarms**: Birds simulate real flock vectors (separation, alignment, cohesion) and butterflies react to mouse movements.
* **Weather Overlay Generator**: Real-time rendering of falling rain, drifting snow, deep fog layers, and random lightning flashes.
* **Zero-Asset Audio Synthesizer (`script.js`)**: Synthesizes organic soundscapes directly inside the browser using noise generators and oscillator networks (Wind, Rain, Night Insects, and Thunder rumbles).
* **Interactive CLI Console**: A command-line terminal interface built into the operator avatar panel at the bottom.

## 🛠️ Setup & Running

### 1. Extract Assets & Starts Server
To copy cached image assets and start the preview server on Windows:
1. Double-click **`copy_assets.bat`** (copies files to `./assets/`).
2. Double-click **`decode_image.bat`** (decodes base64 profile image to `./assets/profile.jpg`).
3. Double-click **`run_server.bat`** (starts server on [http://localhost:8000](http://localhost:8000)).
4. Open your browser and navigate to [http://localhost:8000](http://localhost:8000).

---

## 🌐 Deploy to GitHub Pages

To push to your repository (`https://github.com/veloura07/resume-fun-.git`):
1. Double-click **`deploy_git.bat`**.
2. Log in when prompted. The automated Actions runner (`.github/workflows/deploy.yml`) will automatically publish the website onto GitHub Pages!
